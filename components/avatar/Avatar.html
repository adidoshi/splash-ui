Avatar

Avatars are basically user profile pictures, which are in circular shapes. Avatars can be used in your projects when you want some specific actions to be performed by the user.

Example of rounded avatar

Profile avatars can be created using 'avatar' class and according to preferred size.

Example of square avatar

Adding an extra class of 'square' would change the avatar to square shape.
